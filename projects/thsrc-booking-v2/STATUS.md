## ⚠️ 問題與挑戰

### 1. Wicket Combobox 選擇器

**問題**：鍵盤導航在 Wicket combobox 中無法正常工作

**嘗試的解決方案**：
- ArrowDown + Enter → 無效
- JavaScript 注入 → CLI 環境網絡問題

### 2. 網絡訪問

**問題**：CLI 環境無法訪問高鐵網站（超時）
- Playwright timeout
- curl timeout

**原因**：可能是防火牆或網絡限制

### 3. 驗證碼

**問題**：每次查詢必須輸入驗證碼
- 無法繞過
- 需要 OCR 識別 + 自動填入

## 🔧 解決方案

### 方案 A：瀏覽器手動操作（當前可行）

```
1. 打開瀏覽器訪問高鐵網站
2. 點擊選擇站點
3. 輸入驗證碼
4. 點擊查詢
```

### 方案 B：修復 CLI 腳本

需要解決：
1. 網絡訪問問題
2. Wicket combobox 選擇
