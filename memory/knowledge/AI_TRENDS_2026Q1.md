### 3. Agent 開發觀察

**Tomasz Tunguz 的 9 條觀察**：

1. **多模型協作** - 用 Claude 寫計畫 → Gemini/Codex 批評 → Claude 修正
2. **Agent as Team** - 每個 Agent 有明確角色
3. **人類在環中** - 關鍵決策仍需人類確認
4. **成本控制** - Token 消耗是挑戰
5. **錯誤處理** - 比傳統軟體更難預測
