## 自動化追蹤流程

### Heartbeat 觸發時

```python
# 1. 讀取今日待辦
read("HEARTBEAT.md")

# 2. 讀取當前指標進度
read("memory/knowledge/metrics.md")

# 3. 對照目標差距
calculate_gap(weekly_task, metric_target)

# 4. 產出進度報告
generate_progress_report()

# 5. 記錄到 daily/YYYY-MM-DD.md
write_daily_log()
```
