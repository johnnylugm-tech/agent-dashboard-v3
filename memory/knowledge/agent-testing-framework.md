# AI Agent 測試框架

> 建立 Agent 評測標準

---

## 一、測試金字塔

```
 ┌─────────────┐
 │ E2E 測試 │ 少，關鍵場景
 ┌───────────────┴───────────────┐
 │ 整合測試 │ 中，跨模組
 ┌─────────────────────────────────┴───────────────────┐
 │ 單元測試 │ 多，基本功能
 └─────────────────────────────────────────────────────┘
```

---

## 二、測試類別

| 類別 | 自動化程度 |
|------|------------|
| 單元測試 | 高 |
| 整合測試 | 高 |
| E2E測試 | 中 |
| 效能測試 | 中 |
| 安全測試 | 低 |

---

## 三、核心測試場景

### 單元測試
- 輸入驗證
- 錯誤處理
- 重試邏輯

### 整合測試
- 工具調用
- Agent 協作（Master-Slave, Pipeline）

### E2E 測試
- 完整工作流
- 人類審批流程

---

| 指標 | 目標 | 警告 | 嚴重 |
