# 自動優化響應系統

## 工作流程

### 1. 風格學習 (每次對話)
```
對話 → 分析風格 → 更新 style/boss.json → 調整回應
```

### 2. 需求預測 (每次對話)
```
用戶輸入 → 匹配模式 → 預測下一步 → 主動建議
```

### 3. 反饋收集 (持續)
```
用戶反饋 → 記錄 feedback/boss.json → 調整規則
```

### 4. 迭代優化 (每50條對話)
```
累積數據 → 分析規律 → 更新 response_rules.json
```

## 優化維度

| 維度 | 內容 |
|------|------|
| 長度 | 從 3 行逐漸精煉 |
| 格式 | Markdown / 條列 / 表格 |
| 語氣 | 專業 + 簡潔 + 貼心 |
| 時機 | 主動預測需求 |
