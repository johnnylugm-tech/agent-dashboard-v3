## 1. Model Router 架構

### 核心模組

```
src/
├── classifier.py      # 任務分類 (7 種類型)
├── registry.py        # 模型註冊 (17+ 模型)
├── router.py          # 路由引擎
├── cost_tracker.py    # 成本追蹤
├── trends.py         # 趨勢分析
├── semantic_cache.py # 語意快取
└── failover.py       # 自動 Failover
```

### 路由流程

```
用戶輸入
    │
    ▼
┌────────────┐
│ 任務分類   │ ──→ 識別任務類型
└────────────┘
    │
    ▼
┌────────────┐
│ 預算評估   │ ──→ 根據預算過濾
└────────────┘
    │
    ▼
┌────────────┐
│ 模型選擇   │ ──→ 選擇最佳模型
└────────────┘
    │
    ▼
┌────────────┐
│ API 調用   │ ──→ 執行任務
└────────────┘
    │
    ▼
┌────────────┐
│ 成本記錄   │ ──→ 記錄成本
└────────────┘
```

## 2. Agent Quality Guard 架構

### 核心模組

```
src/
├── analyzer.py      # AST 靜態分析
├── scorer.py        # 評分計算
├── llm_judge.py     # LLM 判斷
├── git_hook.py      # Git Hook
└── reporter.py      # 報告生成
```

### 評估維度

| 維度 | 權重 | 檢查項目 |
|------|:----:|----------|
| 正確性 | 30% | 語法、邏輯 |
| 安全性 | 25% | 漏洞、注入 |
| 可維護性 | 20% | 結構、命名 |
| 效能 | 15% | 複雜度 |
| 測試覆蓋 | 10% | 單元測試 |

## 3. Agent Monitor 架構

### 核心模組

```
src/
├── unified_monitor.py  # 統一監控
├── health_score.py     # 健康評分
├── cost_tracker.py    # 成本追蹤

```

### 監控指標

| 指標 | 說明 |
|------|------|
| 健康評分 | 0-100 |
| Token 消耗 | 即時追蹤 |
| 成本分析 | 每日/每月 |
| 趨勢報告 | 趨勢預測 |

## 5. 錯誤處理

### L1-L5 分類

| 等級 | 類型 | 處理 |
|------|------|------|
| L1 | 輸入錯誤 | 立即返回 |
| L2 | API 錯誤 | 重試 3 次 |
| L3 | 執行錯誤 | 降級處理 |
| L4 | 系統錯誤 | 熔斷保護 |
| L5 | Failover | 切換備援 |
